# NewsApp

**NewsApp** is a Flutter application that provides a seamless news reading experience. It fetches articles from online sources and displays them in a clean, user-friendly interface.

---

## Project Overview

- **Name:** NewsApp  
- **Description:** A Flutter project for reading and managing news content.  
- **Version:** 1.0.0+1  
- **Flutter SDK:** >=3.1.0 <4.0.0  

---

## Features

- Fetch and display news articles dynamically.
- Smooth carousel slider for featured news.
- Cached network images for better performance.
- Pull-to-refresh and infinite scrolling.
- Shimmer loading effects for enhanced UX.
- Webview support for reading full articles.
- State management using **Flutter BLoC**.
- Easy comparison of states with **Equatable**.
